# Style and Context Reference Guide

## Style Categories

### Core Styles
1. **Professional**
   - Real-world: Corporate, Business Casual, Executive
   - Character: Noble, Royal, Aristocratic
   - Key Elements: Tailored fits, structured pieces, refined details
   - Personality: Confident, authoritative, polished

2. **Casual**
   - Real-world: Everyday, Weekend, Smart Casual
   - Character: Commoner, Citizen, Everyday
   - Key Elements: Comfortable fabrics, relaxed fits, versatile pieces
   - Personality: Approachable, authentic, relatable

3. **Formal**
   - Real-world: Black Tie, White Tie, Cocktail
   - Character: Court, Ceremonial, Gala
   - Key Elements: Luxurious fabrics, elegant silhouettes, refined details
   - Personality: Sophisticated, distinguished, elegant

4. **Fantasy**
   - Subcategories: Medieval, Steampunk, Cyberpunk, High Fantasy
   - Key Elements: Period-specific details, magical elements, world-specific materials
   - Personality: Mystical, otherworldly, enchanting

5. **Historical**
   - Subcategories: Victorian, Renaissance, Art Deco, 1920s
   - Key Elements: Period-accurate details, historical references, authentic materials
   - Personality: Time-specific, culturally accurate, period-appropriate

6. **Bohemian**
   - Real-world: Boho Chic, Hippie, Artsy
   - Character: Wanderer, Free Spirit, Artist
   - Key Elements: Flowy fabrics, eclectic patterns, natural materials
   - Personality: Free-spirited, artistic, unconventional

7. **Minimalist**
   - Real-world: Scandinavian, Japanese, Modern
   - Character: Zen, Ascetic, Purist
   - Key Elements: Clean lines, neutral palette, functional design
   - Personality: Organized, intentional, sophisticated

8. **Glamorous**
   - Real-world: Hollywood, Red Carpet, Luxury
   - Character: Diva, Star, Celebrity
   - Key Elements: Shimmering fabrics, bold details, statement pieces
   - Personality: Confident, dramatic, attention-grabbing

9. **Edgy**
   - Real-world: Punk, Rock, Alternative
   - Character: Rebel, Outlaw, Anti-hero
   - Key Elements: Bold patterns, unconventional cuts, statement pieces
   - Personality: Rebellious, confident, individualistic

10. **Romantic**
    - Real-world: Feminine, Soft, Dreamy
    - Character: Lover, Dreamer, Idealist
    - Key Elements: Delicate fabrics, soft colors, flowing silhouettes
    - Personality: Gentle, dreamy, elegant

## Context Categories

### Real-World Events
1. **Business Meeting**
   - Formality: High
   - Key Considerations: Industry, Company Culture, Time of Day
   - Style Recommendations: Professional, Minimalist

2. **Wedding**
   - Formality: Varies
   - Key Considerations: Time of Day, Venue, Role
   - Style Recommendations: Formal, Romantic, Glamorous

3. **Date**
   - Formality: Varies
   - Key Considerations: Activity, Venue, Time
   - Style Recommendations: Romantic, Casual, Glamorous

### Character Contexts
1. **Court Setting**
   - Formality: High
   - Key Considerations: Rank, Status, Role
   - Style Recommendations: Formal, Historical, Noble

2. **Adventure**
   - Formality: Low-Medium
   - Key Considerations: Environment, Activity, Period
   - Style Recommendations: Practical, Period-appropriate, Functional

3. **Ceremonial**
   - Formality: High
   - Key Considerations: Culture, Tradition, Significance
   - Style Recommendations: Formal, Cultural, Ceremonial
